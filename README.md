# 🏠 Sistema de Financiamento Imobiliário

## 📋 Descrição
Simulador de Financiamentos desenvolvido durante a disciplina de Fundamentos da Programação
Orientada a Objetos. O sistema permite simular financiamentos de diferentes tipos de imóveis
(Apartamento, Casa e Terreno), calculando parcelas mensais e total a pagar, além de salvar os 
dados em arquivo de texto e serializado.

## 🛠️ Tecnologias Utilizadas
- Java

## ⚙️ Funcionalidades
- Financiamento de Apartamento, Casa e Terreno.
- Persistência de dados em arquivo.txt e serializado.

## 📁 Estrutura do Projeto
- `modelo/` → Classes de negócio.
- `util/`   → Classes utilitárias.
- `main/`   → Ponto de entrada.
